# USSDGen

Générateur de codes USSD des opérateurs de téléphonie mobile de Côte d'Ivoire.

## Requirements
```
Node ^16.x
```

## Installation
```
git clone https://github.com/eliseekn/ussd-gen.git
```

## Features
- Work Offline
- Secure (no sensitive data stored)
- React Native
- React Native Paper (UI)
- Redux Toolkit
- Typescript
